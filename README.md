# Odin Where's Waldo

A web-based "Where's Waldo" game built as a solution for [The Odin Project assignment](https://www.theodinproject.com/lessons/nodejs-where-s-waldo-a-photo-tagging-app).
The goal of the project is to display a large image containing hidden characters
and allow users to click and identify their locations.
The app records the time taken to find all characters and provides simple feedback.

## Features

- Click-to-select character detection
- Timer that tracks completion time
- Basic validation for correct/incorrect selections
- Simple UI focused on core functionality

## Scripts

- `npm install`: Installs the project's dependencies.
- `npm start`: Starts the development server.
- `npm test`: Runs all the project's tests.
- `npm run build`: Builds the project.
